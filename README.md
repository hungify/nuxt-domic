- [Tailwindcss](https://tailwindcss.nuxtjs.org/) - a utility-first CSS framework for rapidly building custom user interfaces.
- [Pinia](https://github.com/vuejs/pinia) - intuitive, type safe, light and flexible Store for Vue.
